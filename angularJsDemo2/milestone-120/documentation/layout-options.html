Layout Options

Add the following css classes to <div class="app"> to get different layouts

  • .offcanvas - offcanvas sidebar panel.
  • .expanding - expanding sidebar panel.
  • .fixed - main panel header is fixed to top.
  • .boxed - activates boxed layout.
  • .static - static sidebar panel that scrolls with the rest of the page.

For angular you can change your default layout options from app.main.js

Toggle API (HTML version only)

Use data-toggle="expanding" to toggle small sidebar state on click

Use data-toggle="sidebar" to toggle the offscreen / horizontal navigation. .
